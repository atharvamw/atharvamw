## 🚀 Featured Projects

### 🌾 AutoDry — Automation in Food Drying
> *Research-grade • Hardware + AI + Full Stack*

An end-to-end automated food-drying system targeting small, mid, and large-scale enterprises — reducing time loss, weather dependency, and inconsistency in traditional drying processes.

- 🔩 **Hardware:** Raspberry Pi + Pi4j Library for automated temperature & airflow control
- 📱 **Mobile App:** Real-time monitoring & control (Android/Java)
- 🧠 **AI:** Image recognition using the CLIP pre-trained CNN model
- 📄 **Published:** IJSREM, April 2025 · DOI: [10.55041/IJSREM45052](https://doi.org/10.55041/IJSREM45052)
- 🏆 **Won:** NirmitiVedh (DY Patil), FusionX (Navsahyadri), Innovista (Trinity) — Runner's Up at Tech Titans & Kaushalyasetu

`Java` `Python` `JavaScript` `CLIP` `Firebase` `Raspberry Pi` `Sensors`

[![Repo](https://img.shields.io/badge/View%20Repo-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/atharvamw/AutoDry)
